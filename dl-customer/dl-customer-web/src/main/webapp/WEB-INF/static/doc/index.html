返回码说明:

返回参数 返回参数说明
code 返回代号：0访问成功，－1～－99 各类错误信息
cmd 成功/失败信息
values 接口需求的具体数据，键值对列表

请求header:

请求参数 数据类型 请求参数说明
client String 客户端类型：'android', 'ios'
token String 用户32位登录口令
uuid String “设备id”//ios设备id android不填
imei String “设备id”//android设备id iOS不填

pagesize string 页面总数

	        “total” : “活动总数”,
			“activities” : [
				{
					“id” : “活动主键id”,

					“processid” : “活动进程id”，
					“duration” : “活动进程持续时间”，
					“venue” : “活动地点”，
